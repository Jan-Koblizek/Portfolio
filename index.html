RTS Pathfinding

×

The navigation of groups of units in real time strategy games (RTS) was the topic of my master's thesis. My algorithm created a flow network over the map and found alternative paths for the subgroups of units in there. This enabled it to more effectively utilize alternative paths.

The algorithm was implemented using C# and the Unity game engine. I created a testing environment, where the algorithm can be quickly run on different maps and compared against alternative algorithms.
