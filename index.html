RTS Pathfinding

×

The navigation of groups of units in real time strategy games (RTS) was the topic of my master's thesis. My algorithm created a flow network over the map and found alternative paths for the subgroups of units in there. This enabled it to more effectively utilize alternative paths.

The algorithm was implemented using C# and the Unity game engine. I created a testing environment, where the algorithm can be quickly run on different maps and compared against alternative algorithms.

i

Volumetric Clouds

×

I chose real-time volumetric cloud rendering as the topic of my bachelor's thesis. The clouds are modeled using Worley and Perlin 3D noise textures. These are combined together, and each provides a different level of detail. The position of the clouds in the sky is described by a 2D cloud map.

The clouds are rendered using ray-marching. The self-shadowing and the Heneye-Greenstein phase function are computed at each step. The clouds are rendered as a projection onto the sky dome (this means that the player cannot fly through them). To get faster performance, only a part of the pixels is updated for each frame, while others are reused from previous frames (via temporal reprojection).

Magical Pet Shop

The Magical Pet Shop is a shop tycoon in which you craft unique magical animals, send them on expeditions, make a lot of money, and unlock new content! Crafting is divided into several trees, like water, power, fire, and so on. The deeper in the tree, the more powerful the animal it is. LARP Management Application

×

LARP Management Application (LAMA) is a software intended to simplify the organization of live-action role-playing events (LARPs). It simplifies communication between the event organizers and the CPs (character players). The organizers can start actions/scenarios and give the CPs all the necessary information. They can specify the time and the place of the action, the required equipment, and the types of characters that should be there. The application is available both on the desktop and mobile.

We worked on the application in a team of four. I focused on the network communication and encryption. The application was developed for a Software Project course over roughly a year. The code is written in C#, and we use Xamarin for the front end and SQLite as the database.

Mesopotamia

×

Mesopotamia is a 3D hex-based strategy game set in ancient Mesopotamia. The player controls 2 settlements they have to defend against an onslaught of the enemy units. The player can build buildings, train units, and establish new settlements.

I developed the game as homework for the Managed Game Development class. We were supposed to spend roughly 40 hours, but I spent 150+ hours. The game was developed in Unity using C# language. The ground is editable by the game designer and uses a complex shader, which allows it to change its look based on the availability of water in the area. It can also display highlights (for places a unit can move to or the settlement’s boundaries).

Floating River

×

An HLSL shader that mimics the look of flowing water. The shader is inspired by the one created by the Catlike Coding. The water is able to mimic the light absorption and refraction. The reflections are done in screen space. The shader also generates foam at the places where the water intersects with the geometry.
